Aplikasi inventaris barang sekolah menggunakan Framework.

### Fitur

-   CRUD Data Barang
-   Import/export excel barang
-   Print barang (seluruh/individual)
-   CRUD Data BOS (Bantuan Operasional Sekolah)
-   CRUD Data Ruangan
-   CRUD Data Pengguna
-   Pengaturan Profil
